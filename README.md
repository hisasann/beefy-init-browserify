beefyを使ったビルド環境
===============

## 概要

## ツール

* CoffeeScipt
* browserify
  * beefy (auto build & livereload)
  * coffeeify

### インストール方法

#### npmパッケージをインストール

```
npm install
```

### start develop

* livereload and autobuild
  * not created bundle.js
```
npm run dev
```

### build

* create bundle.js
```
npm run build
```
